## Notes for Patent Drawings

When preparing formal patent drawings for submission:

1. **Black and White**: USPTO requires black ink on white paper
2. **No Shading**: Use hatching patterns instead of gray shading
3. **Reference Numbers**: All elements must have reference numerals matching specification
4. **Margins**: 2.5cm top, 2.5cm left, 1.5cm right, 1.0cm bottom
5. **Size**: A4 or US Letter
6. **Quality**: Must be clear enough to photocopy at 2/3 size

These ASCII diagrams can be converted to formal drawings using tools like:
- Microsoft Visio
- Lucidchart
- Draw.io
- Adobe Illustrator
